# 🇱🇰 Explore Sri Lanka

A responsive tourism and travel website showcasing the beauty, culture, food, wildlife, and top destinations of Sri Lanka.

## Features

- Modern responsive design
- Hero section with stunning background image
- Popular Sri Lankan destinations carousel
- Culture and heritage showcase
- Traditional food highlights
- Wildlife information section
- Smooth scrolling navigation
- Mobile-friendly layout

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts (Poppins)

## Project Structure

```
project/
│
├── index.html
├── styles.css
└── README.md
```

## Sections

- Home
- Destinations
- Culture
- Food
- Wildlife

## How to Run

1. Download or clone the repository.
2. Open `index.html` in your web browser.
3. Explore the beauty of Sri Lanka.
